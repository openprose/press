# .prose

OpenProse programs (`.prose` files) for node-rlm automation and self-improvement. These programs run via the OpenProse VM (claude-p) and consume the node-rlm codebase as their subject.

## Contents

- `true-form.prose` — Restores node-rlm to its intended architectural form after drift
- `controlled-burn.prose` — Removes accumulated technical debt and stale artifacts
- `tenet-sync.prose` — Synchronizes TENETS.md with actual implementation practices
- `analyze-trajectories.prose` — Analyzes RLM eval trajectories to surface behavioral patterns and regressions
- `runs/` — Execution artifacts from past .prose program runs (gitignored)
